Tiles Defined. Src.

Constructor for a tilemap

Creates a new tile map. rows and cols are how many different tiles there are in the sprite sheet

_TileWidth and _TileHeight are the size of each individual tile in the sprite sheet. (Typically this works best if they are square for this implementation.)

_mapX, and _mapY are the size of the tilemap. This is the actual number of tiles in the game that the player sees, not how many tiles are in the actual sprite sheet file loaded.

Setup variables

Load the TileMap Image This is the image that will get sliced into smaller subsections of individual tiles.

Create a texture from our surface Textures run faster and take advantage of hardware acceleration

Setup the TileMap array This sets each tile to '0'

Default value is no tile.
